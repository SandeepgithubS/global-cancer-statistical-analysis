# Global Cancer Data Analysis

## Exploratory & Statistical Analysis of Cancer Trends and Risk Factors

# 1. Project Overview:
This project performs an in-depth Exploratory Data Analysis (EDA) and Statistical Analysis on global cancer data to uncover patterns across cancer types, progression stages, demographic variables, and lifestyle risk factors.

The objective was to move beyond surface-level visualization and apply structured statistical techniques to validate relationships and identify meaningful healthcare insights. The analysis combines descriptive statistics, hypothesis testing, correlation analysis, and distribution validation to ensure data-driven conclusions.

Rather than limiting the scope to visualization-based EDA, this analysis incorporates:

* Distribution validation

* Parametric and non-parametric statistical testing

* Correlation strength measurement

* Hypothesis testing with significance validation

* Goodness-of-fit evaluation

The objective is to extract statistically defensible insights from healthcare data and quantify the strength and reliability of observed relationships.

# 2. Business Problem

Healthcare datasets often contain complex interactions between demographic, clinical, and lifestyle variables. Understanding statistically significant relationships can:

* Help identify high-risk groups

* Support early detection strategies

* Improve resource allocation

* Strengthen evidence-based healthcare decisions

* This analysis focuses on extracting measurable, statistically validated insights from structured cancer data.

# 3. Objectives

* Identify distribution patterns across cancer types and stages

* Evaluate relationships between demographic and clinical variables

* Validate statistical significance of observed trends

* Apply hypothesis testing to confirm analytical findings

* Generate structured insights supported by statistical evidence


# 4. Analytical Approach
## A. Data Cleaning & Preprocessing:

* Handled missing values

* Validated data distributions

* Removed inconsistencies

* Standardized categorical variables

* Verified data integrity before analysis

## B. Exploratory Data Analysis (EDA)

* Cancer type frequency distribution

* Stage-wise progression patterns

* Demographic segmentation

* Risk factor comparison

* Distribution analysis of key clinical variables

## C. Statistical Techniques Applied

# Normality Testing:-

## Shapiro-Wilk Test:

* Used to evaluate distribution assumptions

* Determined suitability of parametric vs non-parametric tests

# Correlation Analysis:-

## 1. Pearson Correlation:

* Measured linear relationships between continuous variables

## 2. Spearman Correlation:

* Measured monotonic relationships without normality assumption

## 3. R² (Coefficient of Determination):

* Quantified explanatory strength of relationships

# Group Comparison Testing:-

## 1. ANOVA (Analysis of Variance):

* Compared means across multiple groups

* Applied when normality assumption held

## 2. Kruskal-Wallis Test:

* Non-parametric alternative to ANOVA

* Used when distribution assumptions were violated

# Statistical Validation:-

## 1. Hypothesis testing framework:

* H₀ (Null Hypothesis)

* H₁ (Alternative Hypothesis)

## 2. Evaluated statistical significance using:

* p-values

* Confidence Intervals

* Effect size interpretation

## 5. Key Findings:

* Identified statistically significant relationships between clinical and lifestyle variables.

* Observed demographic patterns influencing cancer progression.

* Detected group-level differences validated using ANOVA and non-parametric testing.

* Confirmed correlation strength using both parametric and non-parametric methods.

* Strengthened analytical findings through hypothesis testing and confidence intervals.
